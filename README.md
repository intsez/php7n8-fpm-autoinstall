# php7n8-fpm-autoinstall
Installation script for php7.4 and 8.0 with some additional modules (Debian/Ubuntu distribution).

### Modules:
php-fpm <br/>
php-mysql<br/>
php-curl<br/>
php-common<br/>
php-mbstring<br/>
php-xmlrpc<br/>
php-gd<br/>
php-xml<br/>
php-cli<br/>
php-zip<br/>
php<br/>
php-imagick

# How to run the script?
Copy/paste into terminal: 

```sh
wget https://raw.githubusercontent.com/intsez/php7n8-fpm-autoinstall/master/php7n8-fpm-autoinstall.sh
chmod +x php7n8-fpm-autoinstall.sh
./php7.x-fpm_setup.sh
